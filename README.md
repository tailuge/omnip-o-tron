Be ready for *all* moves

Based on [lichess' chessground](https://github.com/ornicar/chessground-examples) and [Stockfish](https://github.com/niklasf/stockfish.js)


### Setup

```
nvm use v10.16.3
yarn install
yarn watch 
```
### Test

```
yarn test
```
